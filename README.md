# Banking System

A banking management web application built using ASP.NET MVC, SQL Server, and Entity Framework.

## Features
- User Registration and Login
- Role-Based Authentication
- Account Balance Management
- Deposit and Withdrawal Operations
- Transaction History
- Admin Dashboard
- setting details

## Technologies Used
- ASP.NET MVC
- C#
- SQL Server
- Entity Framework
- Bootstrap

## Project Objective
The purpose of this project is to manage banking operations digitally and provide secure account management for users.
